# Task 8: Simple Sales Dashboard Design

## 📁 Dataset
Used Kaggle Sales Forecasting Dataset: [Kaggle Link](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting)

## 📊 Objective
Build a Power BI dashboard to visualize sales by product, region, and time (month).

## 🔧 Steps Taken
- Loaded the CSV into Power BI
- Created Month-Year column using DAX
- Built:
  - Line chart for Sales over Month-Year
  - Bar chart for Region-wise sales
  - Donut chart for Category-wise sales
  - Map chart for Sales by State
  - Region slicer
- Styled charts and used color formatting
